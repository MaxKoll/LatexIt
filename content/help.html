More about templates

The default template gives a natural, polished look to the LaTeX expressions. It uses the "Palatino" font, and includes several AMS packages to make it easier to insert symbols and specific features. The mathrsfs package allows one to use \mathscr{} to produce English capitals. 

\documentclass{article}
\usepackage{amsmath,amssymb,amsopn}
\usepackage{mathrsfs}
\usepackage{palatino}
\usepackage{mathpazo}
\pagestyle{empty}
\begin{document}
\begin{center}
\LARGE
__REPLACEME__ %this is where your LaTeX expression goes
\end{center}
\end{document}

To insert LaTeX, we first convert dvi to png through ps, then we crop the resulting full-page PNG file to a small image that only contains the formula. The \pagestyle{empty} line removes extra output such as page numbers, so that we just need to crop the blank areas of the big image to get a small image with only the formula. Without this line, there is a page number at the bottom, so cropping doesn't work and a huge image is inserted, which is not what we want!

The __REPLACEME__ word will be replaced with the LaTeX expression you specified. Therefore, it is important you put it in the right place.

We are not using PDFTex but a PS-based production chain, so please keep that in mind when using extra packages (i.e. color).
